# Portfolio Website

A responsive personal portfolio website featuring theme toggle, skills showcase, and project listings.

## Features
- Light/Dark theme toggle with localStorage persistence
- Skills section with card layout
- Projects showcase
- Fully accessible with skip navigation
- Automatically generated last updated date

## Technologies
- HTML
- CSS
- JavaScript
